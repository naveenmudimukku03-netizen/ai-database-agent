# AI Database Agent

## Overview

AI Database Agent is an intelligent conversational AI application that allows users to interact with databases using natural language.

The system works like a ChatGPT-style database assistant where users can ask questions about data without writing SQL queries manually.

The agent understands user requests, analyzes database structure, generates SQL queries, executes them, creates visualizations, and provides meaningful data insights.

---

## Problem Statement

Accessing and analyzing databases usually requires SQL knowledge and understanding of database structures.

AI Database Agent solves this problem by providing a natural language interface that helps users:

* Query databases easily
* Understand database information
* Generate insights automatically
* Visualize data interactively

---

## Objectives

* Build a conversational AI interface for database interaction
* Convert natural language questions into SQL queries
* Connect AI with databases
* Generate dynamic charts and diagrams
* Provide AI-based data explanations

---

## Features

### 1. Chat-Based Interface

A ChatGPT-like interface where users can communicate with the database.

Example:

User:

```
Show me the top 5 products by sales
```

Agent:

* Understands the request
* Creates SQL query
* Fetches data
* Displays results with visualization

---

### 2. Natural Language to SQL

The AI agent converts user questions into SQL commands.

Example:

Input:

```
Find all customers from India
```

Generated SQL:

```sql
SELECT *
FROM customers
WHERE country='India';
```

---

### 3. Database Schema Understanding

The agent automatically retrieves database information:

* Table names
* Column names
* Data types
* Relationships

This helps the AI generate accurate queries.

---

### 4. SQL Query Execution

The generated queries are executed directly on the connected database.

Output:

* Structured data
* Tables
* Query results

---

### 5. Data Visualization

The system automatically generates visual representations:

Supported charts:

* Bar Chart
* Line Chart
* Pie Chart
* Scatter Plot

---

### 6. Diagram Generation

The agent creates:

* Entity Relationship (ER) diagrams
* Database relationship diagrams
* Process flow diagrams

using diagram generation tools.

---

### 7. AI Data Explanation

The agent analyzes results and provides simple explanations.

Example:

"The sales report shows that Product A has the highest revenue compared to other products."

## Agent Tools

### get_schema()

Retrieves database schema details.

Returns:

* Tables
* Columns
* Relationships

### execute_query()

Executes SQL queries generated by the AI agent.

### generate_chart()

Creates dynamic charts from query results.

### generate_flowchart()

Generates ER diagrams and workflow diagrams.

### explain_data()

Provides AI-generated insights from data.

## Installation

Clone the repository:

```bash
git clone YOUR_REPOSITORY_URL
```

Install dependencies:

Backend:

```bash
pip install -r requirements.txt
```

Frontend:

```bash
npm install
```

---

## Environment Setup

Create a `.env` file:

```
OPENROUTER_API_KEY=your_api_key
DATABASE_URL=your_database
```

---

## Running Project

Start Backend:

```bash
python main.py
```

Start Frontend:

```bash
npm run dev
```
